# Subsetting a 'timeDate' object

Extracts or replaces subsets from `"timeDate"` objects.

## Value

an object of class `"timeDate"`

## Examples

``` r
## timeCalendar
tS = timeCalendar()
 
## [
# Subsetting Second Quarter:
tS[4:6] 
#> GMT
#> [1] [2025-04-01] [2025-05-01] [2025-06-01]
   
## [<-
# Replacing:  
```
