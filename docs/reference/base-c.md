# Concatenating 'timeDate' objects

Concatenates `"timeDate"` objects.

## Usage

``` r
# S3 method for class 'timeDate'
c(..., recursive = FALSE)
```

## Arguments

- recursive:

  a logical. If recursive is set to `TRUE`, the function recursively
  descends through lists combining all their elements into a vector.

- ...:

  arguments passed to other methods.

## Value

an object of class `"timeDate"`

## Examples

``` r
## timeCalendar
# Create Character Vectors:
GMT = timeCalendar(zone = "GMT", FinCenter = "GMT") + 16*3600
ZUR = timeCalendar(zone = "GMT", FinCenter = "Zurich") + 16*3600
   
## c
# concatenate and replicate timeDate objects
sort(c(GMT, ZUR))
```
