# Check if dates are business or holidays

Tests if a date is a business day or not.

## Usage

``` r
isBizday(x, holidays = holidayNYSE(), wday = 1:5)
isHoliday(x, holidays = holidayNYSE(), wday = 1:5)
```

## Arguments

- x:

  an object of class `"timeDate"`.

- holidays:

  holiday dates from a holiday calendar. An object of class
  `"timeDate"`.

- wday:

  Specify which days should be considered as weekdays. By default from
  Mondays to Fridays.

## Details

Returns a logical vector of the same length as `x` indicating if a date
is a business day, or a holiday, respectively.

## Value

a logical vector of the same length as `x`
